# KIT85

A full 8085 microprocessor kit simulator for Android.

## Download
[Download Latest APK](https://github.com/forgeVII/KIT85/releases/latest)

## Features
- Full Intel 8085 CPU emulator (all opcodes)
- Real kit layout keyboard
- Memory viewer (64KB hex dump)
- Built-in assembler with syntax highlighting
- Disassembler panel
- Opcode table (List + Grid)
- Number converter (DEC/HEX/BIN/OCT)
- Single step execution
- Register viewer & editor
- Block Move, Memory Compare, String Search, Fill

## How to Use
1. Download the APK from Releases
2. Install on Android (allow unknown sources)
3. Press EXMEM → enter address → NEXT to start

## Built With
- Flutter (no external dependencies)
- Pure Dart
