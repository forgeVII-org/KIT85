# Contributing to KIT85

Thanks for your interest in improving KIT85.

## Development setup

1. Install Flutter SDK (stable channel).
2. Run:

```bash
flutter pub get
flutter analyze
flutter test
```

3. Run app:

```bash
flutter run
```

## Pull request process

1. Create a feature branch from `main`.
2. Keep changes focused and small.
3. Add or update tests when behavior changes.
4. Ensure `flutter analyze` and tests pass.
5. Open a PR with a clear description and screenshots for UI changes.

## Coding guidelines

- Follow existing code style and naming.
- Avoid unrelated refactors in the same PR.
- Keep app behavior stable unless intentionally changing it.

## Reporting issues

Use GitHub Issues with steps to reproduce, expected behavior, and screenshots/logs where useful.
